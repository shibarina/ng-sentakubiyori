# NgSentakubiyori

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.1.2.

## 要件

（機能要件）
・Webページにアクセスしたら、今日が洗濯日和か否かを表示してくれる
・洗濯できるとしたら何時から干せばいいかをレコメンドしてくれる
・今日から３日分の天気が見れる
　→日付は切り替えられる
・GPSでエリア特定
・スマホファーストなUIにすること
　→PCはレスポンシブでよい

（非機能要件）
・Angular Materialを使用
・S3 + Cloud Frontで公開
・ソースコードはGithubにアップロード
